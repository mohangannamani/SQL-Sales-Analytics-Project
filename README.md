# SQL-Sales-Analytics-Project 
📌 Overview
This project focuses on performing advanced sales and customer analytics using SQL Server and Data Warehouse concepts. The analysis helps in understanding business performance, customer behavior, product segmentation, and sales contribution using analytical SQL techniques.
The project is designed to simulate real-world business reporting and analytics scenarios commonly used in data analyst and business intelligence roles.

🚀 Key Features
Monthly Sales Trend Analysis
Customer Segmentation
Product Cost Segmentation
Running Total & Moving Average Analysis
Part-to-Whole Contribution Analysis
Business Performance Reporting

🛠️ Technologies Used
SQL Server
SQL Server Management Studio (SSMS)
Data Warehouse Model
Analytical SQL Queries

📈 Insights Generated
Identified top-performing product categories
Analyzed customer purchasing behavior
Segmented customers into different groups
Tracked sales growth trends over time
Calculated category-wise contribution percentages
Generated business-driven analytical reports

📊 SQL Concepts Applied
Window Functions
Common Table Expressions (CTEs)
Aggregate Functions
Joins
CASE Statements
Date Functions
Analytical Reporting Queries

🎯 Project Goals
The main objective of this project is to strengthen practical SQL skills for:
Data Analytics
Business Intelligence
Reporting
Real-world SQL Problem Solving
